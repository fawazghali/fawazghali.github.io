--- layout: post title: Real-time Stream Processing in the Cloud Challenges and Solutions location: Austin, TX host: Developer Week ---
Abstract:
Real-time stream processing is becoming a critical component in any cloud architecture. The processing times can be measured in microseconds instead of hours or days. However, what if your application requires combining data in motion with data at rest in the cloud? How can you speed up your real-time stream processing?
The talk will focus on speeding real-time stream processing by using concurrency and caching mechanisms in the cloud, it will show why real-time stream processing alone is not enough, and how it can be used in a combination of data at rest to offer more capabilities, and eventually make real-time applications go faster and smarter. The talk will include a demo using the Hazelcast open-source platform and will be available on Github.
